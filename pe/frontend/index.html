① 设备连接

📋 连接指引（首次必读）
  1. 安装 Apple Mobile Device Support：装 iTunes，或从 Microsoft Store 装「Apple Devices」。
  2. iPhone 解锁，用 USB 数据线连接电脑，手机弹窗点「信任」并输入密码。
  3. iOS 16+ 需开启「开发者模式」：若设置 → 隐私与安全性 中看不到该选项，先选中设备后点上方「🔧 开启开发者模式」，按提示在手机弹窗确认并重启；重启后该菜单会出现并已开启。
  4. 点击上方「🔄」刷新，选中设备后点「连接」。

提示：不越狱即可使用，依赖 Apple 官方开发者能力；本程序不收集任何真实位置。
