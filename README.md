LocationMock-iOS-Study（iOS 定位机制研究 Demo）

基于 Windows 平台的 iOS 设备定位模拟研究演示

📌 这是什么？

本项目是一个技术研究性质的演示工具，用于研究 iOS 设备在调试状态下的位置模拟机制：

• Windows 端通过 USB 数据线连接 iOS 设备

• 利用 Apple 官方调试协议与接口

• 向 iOS 设备注入自定义地理位置坐标

• 观察系统及应用对模拟位置的响应行为

它不是：

• ❌ 考勤打卡 / 签到作弊工具

• ❌ 外挂软件或灰色产业工具

• ❌ 可直接无门槛使用的成品应用


------

📌 使用说明（简要）

1. 确保 iOS 设备已开启开发者模式

2. 使用 USB 数据线连接 Windows PC 与 iOS 设备

3. 确认设备被正确识别（驱动正常）

4. 运行调试命令，注入目标坐标

5. 观察 iOS 设备定位变化
